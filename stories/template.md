## Чек-лист качества

### Содержание
- [ ] История конкретная, не абстрактная
- [ ] Есть измеримые результаты (цифры)
- [ ] Понятна ВАША роль (не команды)
- [ ] Есть преодолённые препятствия
- [ ] Есть learning/takeaway

### Подача
- [ ] Укладываюсь в 2-3 минуты
- [ ] Использую "я", а не "мы" для своих действий
- [ ] Нет лишних технических деталей
- [ ] Логичная структура (легко следить)

### Практика
- [ ] Рассказал вслух минимум 3 раза
- [ ] Записал на видео и посмотрел
- [ ] Получил фидбек от кого-то
- [ ] Отработал follow-up вопросы
