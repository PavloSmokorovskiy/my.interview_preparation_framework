# LeetCode Practice Problems

> **Цель:** 50-100 задач для подготовки к coding interview

Задачи отсортированы по категориям и сложности. Начинай с Easy, переходи к Medium.

---

## Как использовать

1. **Решай по категориям** — не перескакивай
2. **Засекай время** — 20 мин Easy, 30 мин Medium, 45 мин Hard
3. **Не подглядывай** — сначала 15 мин сам
4. **Изучай решения** — после решения смотри оптимальные подходы
5. **Повторяй** — через неделю реши снова
